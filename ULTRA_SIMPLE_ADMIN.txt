  MAKE ANY USER ADMIN - ULTRA SIMPLE (2 MINUTES)
═══════════════════════════════════════════════════════════

STEP 1: Get User ID
────────────────────
1. Open: https://console.firebase.google.com/project/proplanet-5987f/authentication/users
2. Click on the user
3. Copy the UID (long ID string)

STEP 2: Create Admin Document  
──────────────────────────────
1. Open: https://console.firebase.google.com/project/proplanet-5987f/firestore/data
2. Click "Start collection"
3. Collection: admins
4. Document ID: [PASTE THE UID HERE]
5. Add 5 fields:
   • email (string) = user's email
   • name (string) = user's name
   • role (string) = admin
   • isActive (boolean) = true
   • createdAt (timestamp) = click "Set"
6. Click "Save"

DONE! User must logout and login again. They're admin now!
